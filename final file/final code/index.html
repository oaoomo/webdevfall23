I'm Abrar, an IDM student at NYU! This website was developed for Intro to Web Dev.

I like old internet things like Windows 95/98 which I tried to make the theme of this website. I also make stuff like films, photos, designs and music (the consquences of being in IDM).

You can find some of my stuff over at the projects page!

In building this website, I did a lot of research on how to make it look like a desktop:

I used icons from https://win98icons.alexmeub.com,

I found Victor Riberio's youtube tutorial (https://www.youtube.com/watch?v=rddE1jKPgWk) which led me to his github in which he built a fake operating system using javascript. That javascript file was used for this final project. https://github.com/victorqribeiro/fos; I also looked to his website (https://victorribeiro.com/) index and css files to understand how certain things worked (specifically the css of his start button and his clock which I used for the date)
Thank you Victor!!

I also found https://simone.computer/#/webdesktops, in which this creator, Simone, collected a bunch of websites that also emulated desktops. I then went through some of them and used developer tools to see how they got certain things to work/look. Designs Photos Videos
